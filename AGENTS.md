这是一个翻译任务，翻译 Ars Magica 的TTRPG规则书到中文。

你需要遵循以下规则：

- 保持 Markdown 结构和原文一致
- 保持术语一致，术语表在 Glossary.md 。
- 不在术语表中的临时专有名词，保持原文不变，或者在后面用括号标注原文，不要生造新词。历史上的人名和地名可以音译。
- 标题使用 `### 中文标题 (English Title)` 的格式
- 适当调整语序以符合中文习惯，保持语言风格流畅自然

## 术语表 (Glossary)

术语表是为了翻译方便。保持说明部分简洁。也可以不包括说明部分。尽量合并多个条目成一条。

## 分块

为了方便翻译有时要把章节拆分成多个文件。

- 例如将 `Faerie/Chapters/04 - Faerie Bestiary.md` 按标题边界拆分，文件名为 `04.01` 至 `04.08`。
- 分块原则：保持原文 Markdown 内容不改写；优先按二级标题拆分，过大的部分再按主要三级标题拆分，使每块适合逐段翻译。

## 翻译流程

1. 将原文 `*/Chapters` 分块，放如译文目录 `*/译稿` 下
2. 逐 section 翻译，把译文放到原文后面，注意 markdown 的空行分段
3. 合并成一个文件，放到 `*/译稿` 下

## 校对

校对时要注意术语是否合理，中文是否流畅自然，以及格式排版是否正确。
